kodama.html ... line画面風、合法的腹話術メモツール

入力した内容がHTML出力されます。
タブキーを使うとtwitter感覚でサクサク使えます。



こだま　を選択すると左側に薄桃で表示され
じぶん　を選択すると右側にグレーで表示されます。
違う色にしたい場合は適当にCSSあたりのソースをいじってください。

再読み込みできれいさっぱり消せます。





Javascriptを使用していますので、記録も残りません。
発言が残ってほしくないけど、
ふとつぶやきたいな　という時にお使いください。
改行およびタグは使用できません。

このプログラムはオープンソースです。
